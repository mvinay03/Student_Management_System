# Student_Management_System

Project Overview :
1. The Student Management System is a console-based Java application developed to manage student records efficiently using Object-Oriented Programming (OOP) principles.
2. This project is designed as a beginner-level application to understand core Java concepts, clean code structure, and layered architecture.
3. The system allows users to add, view, search, update, and delete student records through a menu-driven interface.

Objectives :
1. To understand OOP concepts such as encapsulation, constructors, and object interaction
2. To practice CRUD operations using Java Collections
3. To build a structured and modular Java application
4. To simulate a real-world student record management workflow

Technologies Used :
1. Language: Java
2. Concepts: OOP, Collections Framework
3. Data Structure: ArrayList
4. Interface: Console-based (CLI)

Application Features :
1. Add New Student Details
2. View All Student Data
3. Search Student Data by ID
4. Update Student Data
5. Delete Student Data

How to Run the Project :
1. Clone or download the repository
2. Open the project in any Java IDE (Eclipse / IntelliJ / VS Code)
3. Compile and run MainApp.java
4. Follow the on-screen menu options
